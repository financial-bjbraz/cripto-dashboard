# 01 --- StableCoin Dashboard --- Project Overview

## Objective

Evolve the existing Angular frontend into:

-   Single Page Application (SPA)
-   Progressive Web App (PWA)
-   Integrated with a Backend for Frontend (BFF)
-   Focused on BRLN holders

BRLN is pegged 1:1 with BRL.

## High-Level Architecture

Frontend (Angular SPA + PWA) ↓ BFF (Node 22) ↓ Internal Fiat System +
BRLN Infrastructure
